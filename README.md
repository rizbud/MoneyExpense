## Requirements

- [x] Android SDK
- [x] JDK
- [x] Node
- [x] react-native-cli

## Insallation

Clone this repository
> `$ git clone https://github.com/rizbud/MoneyExpense.git YourAppName`  

Move to your app directory  
> `$ cd YourAppName`  

Run this command  
> `$ yarn install`  

## Runing App

For Android  
>`$ npx react-native run-android`  

For iOS  
>`$ npx react-native run-ios`
